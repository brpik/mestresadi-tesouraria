COMO INSTALAR:

1. Extraia essa pasta.
2. BAIXE A BIBLIOTECA SHEETJS (OBRIGATÓRIO):
   - Acesse: https://cdn.sheetjs.com/xlsx-0.20.1/package/dist/xlsx.full.min.js
   - Salve o arquivo como 'xlsx.full.min.js' DENTRO desta mesma pasta.

3. INSTALAR NO CHROME:
   - Abra chrome://extensions
   - Ative o "Modo do desenvolvedor" (canto superior direito).
   - Clique em "Carregar sem compactação".
   - Selecione esta pasta.

4. DADOS DE TESTE:
   - Use o arquivo dados_exemplo.txt para criar seu Excel.
   - Salve como .xlsx mantendo as abas IRMAOS e PAGAMENTOS.
